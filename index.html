מפרט JSON
כל השדות הנתמכים בwebhook
שדהחובהערכים / תיאור
tickerחובהMNQ, ES, AAPL, BTCUSD, MNQU2025
actionחובהbuy / sell / exit / cancel / add
sentiment—bullish / bearish / flat
signalPriceמומלץמחיר בעת אלרט. משמש לחישוב TP/SL יחסיים, risk sizing, ו-slippage
orderType—market / limit / stop / stop_limit / trailing_stop
limitPrice—נדרש ל-limit / stop_limit
stopPrice—נדרש ל-stop / stop_limit
trailAmount—$ מהwatermark (trailing_stop)
trailPercent—% מהwatermark (trailing_stop)
quantity—מספר (default 1)
quantityType—fixed_quantity / dollar_amount / amount_per_position / risk_dollar_amount / risk_per_position / percent_of_equity / percent_of_position
takeProfit—{percent} / {amount} / {limitPrice}
stopLoss—{type: stop/stop_limit/trailing_stop, percent/amount/stopPrice/trailAmount/trailPercent}
timeInForce—day / gtc / opg / cls / ioc / fok
time—ISO-8601 timestamp (למדידת latency)
interval—timeframe של האלרט (1m, 5m וכו׳)
test—true = לאמת ולתעד אבל לא לבצע
cancel—true = בטל pending orders לפני entry
ignoreTradingWindows—true = להתעלם מ-trading window של ה-strategy
extendedHours—true/false (stocks/options)
delay—שניות (max 30)
extras—אובייקט חופשי לmetadata

📖 מדריך מלא — TradingView Alert Setup
  1. פתח את הגרף שלך ב-TradingView והוסף את ה-Pine Script (אם יש לך).
  2. לחץ על Alerts → Create Alert (או ⌥A).
  3. בשדה Condition בחר: שם ה-Pine script או `alertcondition` שהגדרת (למשל "TradersPost Buy").
  4. בטאב Notifications: הפעל Webhook URL והדבק את ה-URL שבשלב 3.
  5. בשדה Message: הדבק את ה-Alert Message מ-שלב 2.
  6. לחץ Create. ה-alert פעיל.
  7. Tip: להרצה חוזרת בכל bar — בחר "Once Per Bar Close" ב-Options.
⚠ TradingView alerts יכולים לקחת 2-45 שניות להגיע. זה מגבלה של TradingView, לא שלנו.

🚀 Setup Worker — פתרון חיבור Tradovate
🤔 למה זה צריך Worker?
הדפדפן חוסם קריאות ישירות ל-Tradovate API מסיבות אבטחה (CORS policy).
TradersPost עוקפים את זה כי הם Partner רשום של Tradovate עם OAuth.
הפתרון שלנו: Cloudflare Worker (חינם) שרץ ברשת של Cloudflare — בלי CORS.
⚡ אוטומטי — 30 שניות (מומלץ) הכי מהיר
האפליקציה תפרוס את ה-Worker אוטומטית עבורך — יוצרת KV namespace, מעלה קוד, מחברת הכול, ומחזירה URL מוכן לשימוש.
אתה רק צריך API Token חד-פעמי מ-Cloudflare.
1 אין לך חשבון Cloudflare? הירשם (חינם, בלי כרטיס)
🔗 Sign-up
2 צור API Token
לחץ על הכפתור → בחר template "Edit Cloudflare Workers" → Continue → Create Token → העתק
🔑 צור Token ב-Cloudflare
3 הדבק את ה-Token כאן
📋 הגדרה ידנית (אם האוטומטי לא עובד — 10 דקות)
1 פתח Cloudflare + צור Worker חדש
Dashboard → Workers & Pages → Create → Hello World template → שם: profitbot
2 העתק את קוד ה-Worker + הדבק + Deploy
3 הוסף KV Namespace
KV → Create Namespace: PROFITBOT_KV
Worker → Settings → Variables → KV Bindings → Variable: PROFITBOT_KV
4 הדבק URL כאן
